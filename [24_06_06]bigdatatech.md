# 8. 크롤러 유지보수와 운용
## 3. 관리 화면 사용
### 2. peewee와 flask-admin 사용 
5. 관리 화면을 열고 브라우저에서 접근
   - 다음 명령어를 사용해서 관리 화면 실행
   - ```cmd
     $ python flask_admin_server
     ```
   - 브라우저에서 다음 주소로 접근
   - `http://127.0.0.1:5000/admin/publisher`
   - 데이터베이스 publisher 테이블에 등록된 출판사 목록 출력.
   - 출력된 화면의 위키북스 데이터의 Is_Active를 보면 출판사 데이터의 활성화 상태를 나타내는 아이콘이 표시되어 있음. 이어서 '1'이라는 출판사ID 가 출력된 부분의 왼쪽 열을 보면 연필 모양의 아이콘과 휴지통 모양의 아이콘이있음. 연필 모양의 아이콘을 클릭하면출판사 정보의 수정 화면으로 이동
   - Is Active 오른쪽에 있는 체크박스를 클릭해서 체크를 해제, [Save]버튼 클릭.
   - 출판사 목록 화면에서 위키북스의 데이터를 보면 Is Active 열이 비활성화를 나타내는 아이콘으로 바뀐 모습을 볼 수 있음.
   - 출판사 데이터가 낭는 화면 위를 보면 List,Create,Add Filter, With Selected 탭이 있음. [Create] 버튼을 클릭 시 출판사 데이터의 생성 화면이 나옴.
   - 새로운 데이터를 적당히 입력해서 추가, 이 때 [Is Active]의 오른쪽에 있는 체크박스를 클릭해서 체크. [Save] 버튼 클릭
   - 출판사 목록 화면으로 자동으로 복귀, 목록을 보면 데이터가 추가되었다는 걸 알 수 있음. 이번에는 [Add Filter]클릭
   - 메뉴가 열리면 [Name] 선택.이렇게 하면 출력된 contains 옆에 '위키북스'라고 입력 후 [Apply]버튼을 클릭.
   - 출판사 이름으로 '위키북스'가 포함된 데이터만 필터링 되어 출력. [Reset Filters] 버튼 클릭 시 필터 적용 전 상태로 복귀
   - flask-admin을 사용하면 브라우저에서 데이터베이스 테이블의 데이터를 확인, 생성, 변경, 제거할 수 있음. 동작을 모두 확인했다면 터미널로 복귀, [Ctrl] + [C]키 눌러 flask-admin 종료.
