# 3. 크롤러 및 스크레이핑 개발 환경 준비와 파이썬 기본
## 3. 파이썬 기초
### 1. 인터렉티브 셸과 스크립트 파일 실행
1. 인터렉티브 셸 
   - 파이썬의 대화하듯 코드를 실행하는 기능
   - 파이썬 언어를 공부하거나 최신 기능을 테스트해보고 싶을 때 편리한 기능
   - 매개 변수 없이 `python` 명령어 실행 시 인터렉티브 셸 실행 가능
   - 인터렉티브 셸에서 `print("Hello World")`를 입력하고 [Enter]를 누르면 Hello World 출력
   - 인터렉티브 셸을 종료하고 싶을 때는 `exit()` 입력 또는 [ctrl]+[D] 입력
2. 스크립트 실행
   - 인터렉티브 셸이 편리하긴 하지만 일반적으로 파이썬 소스 코드가 들어있는 파일(스크립트 파일)을 지정해서 실행하는 경우 많음. 파이썬 스크립트 파일의 확장자는 .py이고 앞서 사용한 Hello World 명령어를 py 파일로 저장후 실행.
   - 스크립트 파일 실행을 위해서는 다음과 같이 명령어 실행
   - `python <파일 이름>`
   - 현재 디렉터리에 있는 위 Hello World 출력 스크립트 파일을 실행하려면 다음과 같이 입력. ./의 경우 현재 디렉터리를 명시적 지정
   - ./ 를 붙이지않아도 현재 디렉터리에 해당 파일이 있다면 파일 이름만으로 실행 가능
   - `python ./HelloWorld.py`
### 2. 코드 작성 방법
1. 문장 구분
   - 문장을 구분할 때는 줄 바꿈 또는 세미콜론(;)을 사용,
   - 아래 두 코드는 같은 코드임
```python
    a=5
    b=10
    print(a+b)
```
```python
    a=5; b=10; print(a+b)
```
   - 한 줄에 여러 문장을 입력하면 읽기가 굉장히 어려워지므로 특별하 이유가 아니라면 줄 바꿈으로 문장 구분하는 것이 좋음
   - 하나의 문장이 너무 길어서 중간에 줄 바꿈을 하고 싶다면 문장 중간에 역슬래시(\)을 넣고 줄 바꿈
```python
    a = 2 + 4 + 6 + \
    16 + 32 + 64
    print(a)
```
1. 주석
   - 주석은 '#'을 입력, 이후 부분에 입력
```python
    # 초기화하기
    min = 5 # 최솟값
    max = 10 # 최댓값
```
   - 여러 줄 주석은 3개의 큰따옴표 또는 3개의 작은 따옴표로 감싸서 입력
```python
    a = 5
    """
    여러 줄을 
    주석으로 처리
    """
    b = 10
```
   - 원래 3개의 큰따옴표 또는 작은 따옴표로 감싸면 여러 줄 문자열을 생성.
   - 파이썬은 여러 줄 주석 구문이 없지만 코드 중간에 있는 문자열은 실행에 아무런 영향도 주지 않는 사양이 있어 주석으로 사용 가능
