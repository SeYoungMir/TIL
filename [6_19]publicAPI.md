## 공공데이터 포털 사이트 이용방법
[공공데이터 포털(data.go.kr/index.do)](data.go.kr/index.do)
### 사용하기
- 회원가입
- 로그인
- 데이터 활용신청 승인
### 사용 데이터 
- 신청 된 데이터 검색
- 파일데이터 확인
- 사용 데이터
  - '도로교통공단_보행자무단횡단사고다발지역정보서비스'
### 서비스 정보
- 데이터 포멧
- 엔드 포인트
- 보통은 일반 인증키(Encoding)을 사용
- 활용 신청 상세기능 정보
  - 미리보기가 가능함
    - 이 경우 따로 입력할 필요 없이 자동으로 입력되어있음
    - 테스트를 위해 요청한 미리보기 주소를 복사.
- 상세설명 항목
- 요청변수
  - 항목 구분 및 항목 크기, 설명 등
- 출력결과
  - 결과 항목 예시
  - 결과 항목 코드 , 크기 등등
- 활용 정보
  - 기입한 목적과 내용 확인가능
- 라이센스 표시
